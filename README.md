# DS1-Group-Project
This project will explore a 3 million+ dataset of Computer Science papers, including the title, authors, referrences, citations, venue, abstract, and year published.

## Running the code:
Have data json files in the same directory as the jupyter notebook.

## Versions:
- Python 3.13.5
- Pandas 2.3.2
- pyarrow
- joblib
- matplotlib
- seaborn
- scikit-learn
- scipy
